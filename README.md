# LuckySlot

Lucky Slot For Fun ^\_^ (Version 0.1)

## Installation

```
- composer install
- copy .env.example to .env
- setup local database on your machine
- php artisan key:generate
- php artisan migrate
- npm install
- npm run dev
- php artisan serve
```

## Version Used

```
- node version 18.17.0
- npm version 10.2.5
- php version 8.1
- laravel version 10.10
```
